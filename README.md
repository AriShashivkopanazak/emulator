## installation

Dependencies

* Simple DirectMedia Layer (SDL) 2.0 Library

* Gnu Compiler Collection (GCC)

* C Language (clang)

* X server (xorg)

``` bash
git clone https://github.com/AriShashivkopanazak/emulator.git
cd emulator
make
sudo make install
```
